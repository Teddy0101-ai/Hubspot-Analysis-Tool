⚠️ Important: the mapping file (between Step 4 and Step 5)

Step 4 creates the Master Panel mapping file at Output/04/Master Panel SG Hubspot Email - BY Publication.xlsx. It has a Campaign Mapping tab with two control columns you may want to edit before Step 5:

  • MUTE OR NOT — set to 1 to completely exclude a campaign; blank or 0 keeps it.
  • Into All Publications or not? — 1 (or blank) includes it in the "0 All Publications" & loyalty tables; 0 keeps it only in its own publication table.

A ready-to-use blank template of this file is included in the repo under samples/Master Panel - TEMPLATE.xlsx so you can see the exact column layout. Step 4 regenerates the real one from your data each time it runs.

🧩 Chrome Extension — HubSpot Bulk Recipient Export (ZIP)

Select multiple marketing emails in HubSpot and download all their recipient lists as XLSX in one named ZIP.

⬇ Download extension (.zip)

Install it once in Chrome (or Edge) and a “Download selected (ZIP)” button appears on the HubSpot email management page.

  1. Download the ZIP above and unzip it to a permanent folder (don’t delete it later — Chrome loads it from there).
  2. Open chrome://extensions (copy-paste into the address bar).
  3. Turn on Developer mode (toggle, top-right).
  4. Click “Load unpacked” and select the unzipped HubSpot-Bulk-Recipient-Export folder (the one containing manifest.json).
  5. Go to HubSpot → Marketing ▸ Email, tick the emails you want, and click “Download selected (ZIP)”.

Works on https://app.hubspot.com/email/<portal>/manage/…. Only needs the downloads permission. To share with a colleague, just send them the ZIP and these 5 steps — they don’t need this analysis tool running to use the extension.
